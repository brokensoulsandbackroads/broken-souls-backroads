BROKEN SOULS & BACKROADS V2
============================

This version is GitHub Pages ready.

UPLOAD THESE ITEMS TO GITHUB
----------------------------
assets
index.html
merch.html
style.css
script.js
README.txt

DO NOT upload the ZIP file itself.
Unzip it first, then upload the contents.

GITHUB PAGES SETUP
------------------
1. Create a new public repository.
2. Upload all files and folders from this extracted folder.
3. Click Settings.
4. Click Pages.
5. Source: Deploy from a branch.
6. Branch: main.
7. Folder: /root.
8. Save.
9. Wait 1-3 minutes.
10. Your site will appear at:
   https://YOURUSERNAME.github.io/REPOSITORY-NAME/

IF IMAGES DO NOT SHOW
---------------------
Check that the assets folder is visible in your repository.
It must contain files such as:
- hero.jpg
- backroads-motel.jpg
- tee.jpg
- merch-hero.jpg

CUSTOM DOMAIN LATER
-------------------
You can add your domain later from Settings > Pages > Custom domain.
